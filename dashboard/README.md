# لوحة تحكم ريف السعودي - Reef Dashboard

## نظرة عامة
لوحة تحكم متكاملة لإدارة طلبات التمويل مع نظام إشعارات عبر البريد الإلكتروني وصفحة تسجيل دخول مدمجة.

## الميزات الرئيسية

### 1. صفحة تسجيل الدخول (Login Page)
- تصميم عصري ومتجاوب مع خلفية متحركة
- دعم كامل للغة العربية واتجاه RTL
- خاصية "تذكرني" للحفاظ على جلسة المستخدم
- رسائل خطأ واضحة وتجربة مستخدم سلسة
- مسار: `/login`

### 2. نظام الإشعارات عبر البريد الإلكتروني
يتم إرسال إشعارات تلقائية للمستخدمين في الحالات التالية:

#### أ. عند إضافة طلب جديد
- يتم إرسال بريد إلكتروني للمستفيد يحتوي على:
  - رقم الطلب
  - القيمة المطلوبة
  - تاريخ التقديم
  - رابط لمتابعة حالة الطلب

#### ب. عند تحديث حالة الطلب
- يتم إرسال بريد عند تغيير الحالة (pending → approved/rejected/completed)
- يعرض الحالة القديمة والجديدة بألوان مميزة
- إذا تمت الموافقة، يعرض القيمة الموافق عليها

#### ج. عند الاستعلام عن طلب
- يتم إرسال بريد تأكيد الاستعلام
- يتضمن تاريخ ووقت الاستعلام

### 3. لوحة التحكم للمشرفين
- عرض إحصائيات الطلبات (الإجمالي، قيد الانتظار، الموافق عليها، المرفوضة)
- جدول لعرض جميع الطلبات مع خيارات البحث والتصفية
- نموذج إضافة طلب جديد مع جميع الحقول المطلوبة
- إدارة كاملة للطلبات (إضافة، تحديث، حذف)

### 4. صفحة التحقق من حالة الطلب
- تتيح للمستخدمين التحقق من حالة طلباتهم
- باستخدام رقم الهوية ورقم الطلب
- مسار: `/check-status`

## إعداد البيئة التطويرية

### المتطلبات المسبقة
- Node.js (أحدث إصدار LTS)
- pnpm (مدير الحزم)
- حساب Gmail أو خادم SMTP آخر

### التثبيت

1. تثبيت الاعتماديات:
```bash
pnpm install
```

2. إنشاء ملف `.env` في المجلد الرئيسي وإضافة الإعدادات التالية:
```env
# إعدادات قاعدة البيانات
DATABASE_URL=file:./dev.db

# إعدادات OAuth (اختياري)
VITE_OAUTH_PORTAL_URL=https://your-oauth-portal.com
VITE_APP_ID=your-app-id

# إعدادات البريد الإلكتروني
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SMTP_FROM=Your Name <your-email@gmail.com>
```

### إعداد Gmail لإرسال البريد الإلكتروني

1. تفعيل التحقق بخطوتين في حساب Google
2. إنشاء كلمة مرور للتطبيق (App Password):
   - اذهب إلى https://myaccount.google.com/apppasswords
   - اختر "Mail" وجهازك
   - انسخ كلمة المرور المكونة من 16 حرفاً
   - الصقها في `SMTP_PASSWORD` في ملف `.env`

### تشغيل المشروع

```bash
# وضع التطوير
pnpm dev

# البناء للإنتاج
pnpm build

# تشغيل نسخة الإنتاج
pnpm start
```

## استخدام API

### نقاط النهاية المتاحة

#### الحصول على قائمة الطلبات (للمشرفين فقط)
```typescript
trpc.requests.list.useQuery({
  search: "بحث اختياري",
  status: "pending | approved | rejected | completed",
  limit: 10,
  offset: 0
});
```

#### إضافة طلب جديد
```typescript
trpc.requests.create.useMutation({
  onSuccess: () => {
    // يتم إرسال بريد إلكتروني تلقائياً
  }
});
```

#### تحديث طلب
```typescript
trpc.requests.update.useMutation({
  onSuccess: () => {
    // إذا تغيرت الحالة، يتم إرسال بريد إلكتروني
  }
});
```

#### الاستعلام عن طلب
```typescript
trpc.requests.inquire.useQuery({
  identityNumber: "رقم الهوية",
  requestNumber: "رقم الطلب"
});
```

## قوالب البريد الإلكتروني

جميع قوالب البريد موجودة في `server/email.ts` وتتضمن:
- تصميم متجاوب باللغة العربية
- ألوان متناسقة مع هوية التطبيق (أخضر وذهبي)
- معلومات واضحة ومنظمة
- روابط للعمل (CTA buttons)

## بيانات الدخول التجريبية

لأغراض الاختبار، يمكنك استخدام:
- اسم المستخدم: `admin`
- كلمة المرور: `password123`

**ملاحظة:** في بيئة الإنتاج، يجب استخدام نظام مصادقة حقيقي مع تجزئة كلمات المرور.

## التقنيات المستخدمة

- **Frontend:** React 19, TypeScript, Tailwind CSS, Radix UI
- **Backend:** Express, tRPC, Zod
- **Database:** SQLite مع Drizzle ORM
- **Email:** Nodemailer
- **Routing:** Wouter
- **Build Tool:** Vite
